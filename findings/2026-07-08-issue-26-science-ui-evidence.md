## Observed Science UI

Playwright opened the isolated Science URL generated by the sandbox daemon. The UI loaded under the virtual local account `virtual@localhost.invalid`, then entered the example project.

Opening the model selector showed a non-empty menu item:

```text
menuitemradio "glm-4.5 Best for scientific rigor"
```

After selecting it, the composer button changed to:

```text
button "Model: glm-4.5"
```

This verifies that the isolated Science selector can see and select the proxy `display_name` for the forced custom OpenAI model.
